The service allows you to apply watermarks to images and download it further.

How to install the project

  • Ruby is required

  • Node.js is required

  1. git clone https://github.com/ImMOrta1/addWatermark.git addWatermark
  2. cd addWatermark
  3. bower install
  4. npm install
  5. gem install sass
  6. gem install compass
  7. gulp

How to use the service

  1. Upload source image;
  2. Upload image for watermark;
  3. Change watermark's position and opacity if it necessary;
  4. Download the result.

Instruments

  • Bower
  • Jade
  • Sass
  • Compass
  • Gulp
  • HTML5
  • CSS3
  • JavaScript
  • PHP
